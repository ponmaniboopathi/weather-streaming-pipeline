
## Table: weather-table

| Column Name     | Type     | Description                     |
|----------------|----------|---------------------------------|
| name           | string   | City name                       |
| region         | string   | State/region                    |
| country        | string   | Country name                    |
| lat            | real     | Latitude                        |
| lon            | real     | Longitude                       |
| localtime      | datetime | Time of data collection         |
| temp_c         | real     | Temperature in Celsius          |
| is_day         | int      | Day (1) or Night (0)            |
| condition_text | string   | Weather condition description   |
| condition_icon | string   | Weather icon URL                |
